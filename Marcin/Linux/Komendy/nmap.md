# namp
Komenda `nmap` służy do eksploracji sieci i audotywania zabezpieczeń. 

Uzyskujemy informacje o 
1. dostepności sieci, 
2. szczegółowych informacji o IP, 
3. otwartych portach
4. lista hostów
5. skanowanie portów, OS i hostów

**Przykład użycia:**

`nmap www.facebook.com`
