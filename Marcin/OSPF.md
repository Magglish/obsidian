# OSPF
OSPF (Open Shortest Path First) najczęściej używany protokół [[IGP]]. 

1. Protokół dla [[IPv4]], [[IPv6]] oraz [[CIDR]]. 
2. Używany przez [[Gateway]], [[Router]]
3. Używa [[Link state]] routing algorytm.
